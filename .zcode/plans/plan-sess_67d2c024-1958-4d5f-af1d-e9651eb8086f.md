## Fyne v2.8 完整重写计划

### 核心架构

**侧边面板是主窗口内的区域**（不是独立窗口），消除之前所有的多窗口问题。单窗口 master-detail 布局：

```
┌──────────────────────────────────────────┐
│  [标题栏拖拽区]  [+] [↻] [📌] [🔒] [⚙] [×]    │  自定义顶栏（无边框）
├──────────────────┬───────────────────────┤
│                  │  [<] 详情标题          │  面板顶栏（"<" 折叠按钮）
│  待办列表        ├───────────────────────┤
│  ┌─────────────┐ │                       │
│  │ B-12  P1 ▶ │ │  详情/管理/新建内容    │
│  └─────────────┘ │  （可滚动）           │
│  ...             │                       │
│  共 12 项        │                       │
└──────────────────┴───────────────────────┘
```

**Dock 自动隐藏条件**：只在面板**已隐藏** + 非锁定 + 已 dock 到边缘时触发。面板展开时不自动隐藏。

### 各模块设计

**theme/theme.go** — 实现 `fyne.Theme`：Background=#FFF, Foreground=#1A1A1A, Button 深色填充白色文字, Disabled=#BDBDBD, Radius=4dp, TextSize=12。全局统一所有按钮/输入框样式。

**ui/app.go** — App 控制器，持有 Window/List/Panel/Login。布局：登录页→Login；列表页→顶栏(Border顶部) + 列表/面板(HSplit)。`OpenDetail/OpenManage/OpenCreate`→设置面板模式+展开。面板折叠时窗口缩窄。

**ui/titlebar.go** — 无边框自定义顶栏。左侧拖拽区（`Draggable`+`RequestPosition`，避免 Win11 Snap），右侧 [+] [↻] [📌] [🔒] [⚙] [×] 按钮。

**ui/list.go** — `widget.List` 虚拟化列表（内置滚动），每行：code+标题 | 优先级 | 状态 | 操作按钮。排序点击 toggle。行点击→OpenDetail。操作按钮→start/complete（15s+冲突）。选中行高亮。

**ui/panel.go** — 主窗口内右侧区域。三种模式（Detail/Manage/Create）切换内容。顶栏"<"按钮折叠面板。`Show(mode)`/`Hide()`/`IsHidden()`。

**ui/detail.go** — `container.NewScroll` 包裹。查看模式：code+标题+状态/操作+删除/描述/元数据/关联/评论。编辑模式：Entry 表单。冲突用 `dialog.NewConfirm`。评论增删。

**ui/manage.go** — 语言切换/Dock设置/搜索筛选/chip切换/应用/登出。`container.NewScroll` 包裹。

**ui/dock.go** — 独立 goroutine 16ms 轮询，复用 `store/dock.go`。动画用 `RequestPosition`+ease-out。自动隐藏条件：`panel.IsHidden() && !locked && docked && topmost && !dragging`。6px nearEdge + 1000ms grace + 可配置 hideDelay。

**platform/platform.go** — 接口：`SetAlwaysOnTop`/`SetLock`/`MoveWindow`/`WindowGeometry`/`WorkArea`/`CursorPos`/`HideFromTaskbar`。置顶和定位用 `desktop.Window` API，锁定用 RunNative Win32。

**platform/windows/lock.go** — `driver.NativeWindow.RunNative()` 拿 HWND，设置/清除 `WS_EX_LAYERED|WS_EX_TRANSPARENT` + `SetLayeredWindowAttributes`。约 50 行。

**main.go** — `app.New()` + 自定义 theme + 创建无边框窗口 + `SetSystemTrayMenu`（锁定✓/置顶✓/退出）+ `SetCloseIntercept`（dialog 确认）+ dock goroutine + 恢复 Locked/TopMost。

### 技术映射

| 功能 | Fyne API |
|---|---|
| 侧边面板 | 主窗口内 container（非独立窗口） |
| 滚动列表 | `widget.List` 虚拟化 |
| 按钮样式 | `fyne.Theme` 统一 |
| 窗口置顶 | `desktop.Window.RequestAlwaysOnTop()` |
| 窗口定位 | `desktop.Window.RequestPosition(x,y)` |
| 自定义拖拽 | `Draggable` + `RequestPosition` |
| 系统托盘 | `fyne.io/systray` 内置 |
| 模态对话框 | `dialog.NewConfirm` |
| 点击穿透+透明 | `RunNative` → Win32（~50 行） |
| 跨线程 UI | `fyne.Do(func(){})` |
| 无边框 | `SetDecorated(false)` |
| 面板折叠 | 容器可见性 + 窗口 Resize |
